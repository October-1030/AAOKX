========================================
第一步：重启项目（必须！）
========================================

因为我刚修改了代码，你需要重启项目才能看到效果：

方法1（推荐）：
   双击项目文件夹里的：重启项目.bat

方法2（手动）：
   1. 在运行 npm run dev 的命令行窗口按 Ctrl+C 停止
   2. 重新运行：npm run dev
   3. 等待显示 "Ready in X.Xs"

========================================
第二步：打开浏览器
========================================

访问：http://localhost:3000

========================================
第三步：看到效果
========================================

页面加载后你会看到：
- 顶部：ALPHA ARENA 标题
- 右上角：绿点 "CONNECTED"（如果是红点，刷新页面）
- 中间：6种加密货币的市场数据
- 下面：6个AI模型，每个都是 $10,000

========================================
第四步：点击按钮！
========================================

🔵 点击 "Execute Cycle" 按钮
   ⏳ 等待 5-10 秒（AI在思考）
   ✅ 你会看到：
      - 某些模型的账户金额变化了！
      - 出现 "+2.3%" 或 "-1.5%" 的收益率
      - 下方显示持仓信息（绿色框=做多，红色框=做空）

🟢 点击 "Start Trading" 按钮
   - 按钮变成红色 "Stop Trading"
   - 每3分钟自动交易一次
   - 排行榜自动更新

========================================
观察交易效果
========================================

点击几次 Execute Cycle 后，你会看到：

1. 排行榜变化
   - 有些模型赚钱（绿色 +5.2%）
   - 有些模型亏钱（红色 -2.1%）
   - 排名会重新排序

2. 持仓信息
   🟢 绿色框 = 做多（LONG）- 价格上涨赚钱
   🔴 红色框 = 做空（SHORT）- 价格下跌赚钱

   显示内容：
   - SOL 15x LONG（SOL币种，15倍杠杆，做多）
   - Entry: $183.25（入场价格）
   - Stop Loss: $179.38（止损价）
   - Take Profit: $190.58（止盈价）

3. 权益曲线图
   - 滚动到下方
   - 看到彩色的线条
   - 不同颜色=不同AI模型的表现

4. Model Chat（模型对话）
   - 选择不同的AI
   - 查看它们为什么买入/卖出

========================================
常见问题
========================================

Q: 点击后还是没反应？
A:
   1. 确认你重启了项目
   2. 刷新浏览器页面（按F5）
   3. 等待至少10秒
   4. 检查右上角是否显示绿点"CONNECTED"

Q: 所有模型都没有变化？
A: 这是概率问题，再点击1-2次就会看到交易

Q: 为什么有些模型不交易？
A: Claude和Grok设置得比较保守，它们大部分时间在观望

Q: 如何让交易更频繁？
A: 已经调整好了！Gemini有80%概率会交易
